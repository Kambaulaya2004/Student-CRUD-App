STUDENT CRUD APP.
It is a project leveraging kotlin language to create Mobile App for CRUD operation speciaalized for Students operations.
The Project performs 4 features:
a. Creating/ Adding the Student dtails.
b. Updating the existing Students data.
c. Deleting the Student And its details.
d. Reading the available Students data.

The UI of the program, show the User the Lists of available Students who have already added in the system........
